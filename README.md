# -Helpdesk-ticket-simulator
This project simulates a basic It helpdesk ticketing system where users can create,view and manage support tickets. it demonstrates core It support workflow concepts such as issue tracking, categorization, and status management. 


Technologies Used
* Python Coding language 
* VS Code
*  GitHub




     What I learned

   * How ticketing systems work in IT support environments
   * Basic CRUD operations (Create, Read, Update, Delete)
   * Organizing code into functions/classes
   * Simulating real-world helpdesk workflows
